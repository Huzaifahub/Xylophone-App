## ğŸ¯ How to Use
## Start the app:
When you launch the app, you'll see a series of buttons, each labeled as "Music 1", "Music 2", etc.
## Tap to Play: 
Tapping on a button will start playing the corresponding note.
## Pause Functionality:
If you tap on the same button again, the music will pause.
## Switch Notes:
If a note is playing and you tap on another button, the app will switch and play the new note.
